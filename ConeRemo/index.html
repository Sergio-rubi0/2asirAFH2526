Conexión remota

1. Instalar un servidor SSH en Linux

Crear usuarios

sudo adduser sergiorubio

Instalar el servidor SSH

sudo apt install openssh-server

Probar conexión desde Windows (Putty)

login as: sergiorubio
sergiorubio@192.168.10.6's password:
Welcome to Ubuntu 20.04.1 LTS

Probar desde otro Linux

ssh sergiorubio@192.168.10.6

Conexión sin contraseña (clave pública)

ssh-keygen
ssh-copy-id sergiorubio@192.168.10.6
ssh sergiorubio@192.168.10.6

2. Conexión de escritorio remoto en Windows (RDP)

Activar RDP

Ir a Panel de control > Sistema > Acceso remoto y habilitar las conexiones remotas.

Conexión desde cliente RDP

Escribir las credenciales para conectarse a 192.168.10.20
Usuario: alumno

Verificar conectividad con ping

ping 192.168.10.6
ping 192.168.10.1

3. Conexión remota con TeamViewer

Instalación en Windows y Linux

Instalar TeamViewer en ambos equipos y usar el ID de sesión para conectarse.

Conexión desde móvil

Usar la app de TeamViewer para acceder remotamente desde dispositivos móviles.

Opciones avanzadas

  • Modo presentación
  • Transferencia de archivos
  • Control remoto
